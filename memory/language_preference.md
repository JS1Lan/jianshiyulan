All generated projects, apps, and any user-facing content must be in Chinese by default. This includes UI labels, button text, error messages, placeholders, alerts, empty states, navigation labels, and documentation.

**Why:** User explicitly requested: "不管是这个APP还是以后生成的任何成果都要是中文" (whether this app or any future output, everything must be in Chinese).

**How to apply:** When building any project with user-facing text, use Chinese as the primary language. This includes React Native / React apps, web apps, CLI tools, and any other software output. Only use English if the user explicitly requests it for a specific project.
